# 📊 Electronics Sales Dashboard

An AI-powered Electronics Sales Dashboard built using Flask, SQLite, HTML, CSS, JavaScript, and Machine Learning.



📸 Dashboard Preview

![Dashboard](images/dashboard.png)




## 📖 Project Overview

The Electronics Sales Dashboard is a web application for managing electronic sales, products, customers, and orders. It also provides analytics, export functionality, dark mode, and AI-based sales prediction using a Random Forest model.

---

## ✨ Features

- 🔐 Login Authentication
- 📊 Interactive Dashboard
- 🏢 Company Management
- 📦 Product Management
- 👥 Customer Management
- 📋 Order Management
- 📈 Sales Analytics
- 🤖 AI Prediction
- 🔍 Search Companies
- 📤 Export Dashboard Report
- 🌙 Dark Mode

## 🛠 Technologies Used
